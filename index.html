Git Notes

git init - initiate git command

git status - check the current status

git add - add files like html, css, or js

git commit -m "" - put commit changes and its descriptions

git log - history of commits

git checkout /log changes/- grabbing a file and go back to that change like a time machine

git branch - check the current branch youre in

git branch nameofnewbranch - creating a new branch

git remote -v - check the origin you have

git add origin url - add the link from github

git push -u origin main - upload commits on github
